# GETTING STARTED

* Clone repository

```bash
    git clone git@gitlab.asoft-python.com:hoan.hoang/react-training.git
```

* Cd to the root folder

```bash
    cd react-training
```

* Checkout to the branch 

```bash
    git checkout nextjs-practice
```

* Cd to the folder

```bash
    cd nextjs-practice/shady-rhymes
```

* Install the packages

```bash
    pnpm install
```

* Run the practice

```bash
    pnpm dev
```

* Run the storybook

```bash
    pnpm storybook
```

* Run the test

```bash
    pnpm jest
```
